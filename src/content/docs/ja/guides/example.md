# サンプルガイド

SenseCraft HMI サンプルガイドへようこそ！

## クイックスタート

これは、Starlight でドキュメントページを作成する方法を示すサンプルページです。

### 機能

- Markdown 構文のサポート
- 自動サイドバー生成
- 多言語サポート
- レスポンシブデザイン

### 次のステップ

1. このファイルを編集してコンテンツを追加
2. より多くのガイドページを追加
3. サイドバーを設定

## コードサンプル

```javascript
// これは JavaScript コードサンプルです
function hello() {
  console.log("Hello, SenseCraft HMI!");
}
```
