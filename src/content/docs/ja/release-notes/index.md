このページには、SenseCraft HMI プラットフォームの更新記録とバージョン履歴が含まれています。

## v1.0 - 2025-10-24

**再設計されたレイアウトとページリストで簡単に作成**

- ナビゲーションを容易にする新しい UI レイアウト
- ページリスト機能を追加し、Keynote スライドのように複数の UI を作成
- ページライブラリを追加し、すべての UI を保存

**強化された UI エディターとデータ統合**

- 横向きと縦向きのキャンバスレイアウトをサポート
- UI テンプレートのインポートとエクスポートをサポート
- アカウントから SenseCAP のライブデータを直接取得
- デバイスの最終オンライン時間と次回更新時刻を追加

**拡張されたハードウェアサポート**

- ePaper DIY Kit-EE04 をサポート、7.3" と 7.5" ディスプレイに対応
- TRMNL 7.5" (OG) DIY Kit の 7.5" ディスプレイをサポート

**ファームウェアフラッシュと画像処理ツールページ**

- SenseCraft HMI ファームウェアと TRMNL ファームウェアの両方のフラッシュをサポート
- ディザリングツールを追加して画像を処理し、.c ファイルに直接エクスポート

- **ファームウェア管理**：オンラインファームウェアフラッシュと更新
