reTerminal E シリーズ電子ペーパーディスプレイは、強力なコンピューティング能力と電子ペーパーディスプレイ技術を組み合わせており、デジタルサイネージ、情報ダッシュボード、インタラクティブディスプレイに最適です。このシリーズには、異なる表示機能を持つ2つのモデルが含まれています：

#### [reTerminal E1001](https://wiki.seeedstudio.com/getting_started_with_reterminal_e1001/)

reTerminal E1001 は、4レベルのグレースケール機能を備えた 7.5 インチモノクロ電子ペーパーディスプレイを搭載しています。800×480 の解像度により、鮮明なテキストと明瞭な画像を提供しながら、最小限の電力を消費します。モノクロディスプレイは、テキストの読みやすさとバッテリー寿命が優先される用途に最適です。

**主な機能**

- **美しく、すぐに使える電子ペーパーディスプレイ**
- **3ヶ月のバッテリー寿命を実現する超低消費電力**
- **SenseCraft HMI によるノーコード UI 設計とデプロイ**
- **人気のソフトウェアプラットフォームと連携**
- **柔軟なハードウェアとソフトウェアのカスタマイズ**

#### [reTerminal E1002](https://wiki.seeedstudio.com/getting_started_with_reterminal_e1002/)

reTerminal E1002 は、ACeP（Advanced Color ePaper）技術を採用した鮮やかな 7.3 インチ6色電子ペーパーディスプレイを搭載しています。このカラーディスプレイは、電子ペーパー技術の同じ省電力性を維持しながら、複数の色で視覚的に魅力的なコンテンツを提供できるため、視覚的な区別と美的魅力が重要な用途に最適です。

**主な機能**

- **美しく、すぐに使える E Ink® Spectra™ 6色電子ペーパーディスプレイ**
- **3ヶ月のバッテリー寿命を実現する超低消費電力**
- **SenseCraft HMI によるノーコード UI 設計とデプロイ**
- **人気のソフトウェアプラットフォームと連携**
- **柔軟なハードウェアとソフトウェアのカスタマイズ**
