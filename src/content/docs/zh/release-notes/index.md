本页面包含 SenseCraft HMI 平台的迭代记录和版本历史。

## v1.0 - 2025-10-24

**重新设计的布局和页面列表，更易于创建**

- 全新的 UI 布局，便于导航
- 添加页面列表功能，像 Keynote 幻灯片一样创建多个 UI
- 添加页面库，保存您的所有 UI

**增强的 UI 编辑器和数据集成**

- 支持横向和纵向画布布局
- 支持 UI 模板导入和导出
- 直接从您的账户获取 SenseCAP 实时数据
- 添加设备上次在线时间和下次刷新时间

**扩展的硬件支持**

- 支持 ePaper DIY Kit-EE04，兼容 7.3" 和 7.5" 显示器
- 支持 TRMNL 7.5" (OG) DIY Kit 配 7.5" 显示器

**固件刷写和图像处理工具页面**

- 支持刷写 SenseCraft HMI 固件和 TRMNL 固件
- 添加抖动工具以处理图像并直接导出为 .c 文件

## v0.6

### 初始版本
- **设备支持**：reTerminal E1001 和 E1002
- **页面类型**：支持单独的图库、画布、RSS 和网页
- **固件管理**：在线固件刷写和更新
